2.  内外：放弃外在锚定，稳定心锚，从我出发，不战不败。认识自己，听从我心，追随我感；爱我所爱，行我所行，无问东西。

3.  更好：活出我的快乐和痛苦，追求自由、平等、幸福的人生和环境；找到组织和共同体，成为自己，爱好一个人，做好一件事。
